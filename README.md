# Rust Lottery

This is a demo app to play around with Rust and multi thread

## Installation

* Copy this repo
* Build the project with Cargo for dev or production

or

* Run the project with Cargo

```bash
cargo build

cargo build --release

cargo run
```

## Usage

```bash
./target/release/lottery
```
